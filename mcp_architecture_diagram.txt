                         KEY CHARACTERISTICS
═══════════════════════════════════════════════════════════════════════════════

✓ Client-Server Architecture
✓ JSON-RPC 2.0 Protocol
✓ Two-Layer Design (Data + Transport)
✓ Multiple Transport Options (STDIO, HTTP)

✓ Dynamic Discovery (*/list methods)
✓ Real-time Notifications
✓ Flexible Authentication (OAuth, Bearer tokens, API keys)
