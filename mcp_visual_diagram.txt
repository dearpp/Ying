║                                    KEY POINTS                                     ║
╚═══════════════════════════════════════════════════════════════════════════════════╝

✓ Client-Server Architecture: Host manages multiple clients, each connected to a server
✓ Two-Layer Design: Transport layer (outer) + Data layer (inner)
✓ JSON-RPC 2.0: Standard protocol for all communication
✓ Dynamic Discovery: Use */list methods to discover primitives
✓ Flexible Transport: Local (STDIO) or Remote (HTTP)
✓ Rich Primitives: Tools, Resources, Prompts (server) + Sampling, Elicitation, Logging (client)
✓ Real-time Updates: Notification system for dynamic changes
